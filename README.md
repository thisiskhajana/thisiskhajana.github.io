## 🚀 Getting Started

### Local Development

```bash
# Clone the repository
git clone https://github.com/thisiskhajana/thisiskhajana.github.io.git
cd thisiskhajana.github.io

# Option 1: Open directly in browser
open index.html

# Option 2: Python local server (recommended)
python -m http.server 8000
# Visit http://localhost:8000
```

### Deployment

The site is hosted on GitHub Pages and automatically deploys on push to `main`:

```bash
git add .
git commit -m "Update portfolio"
git push origin main
```
