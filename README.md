## 🚀 Getting Started

### Local Development

```bash
# Clone the repository
git clone https://github.com/thisiskhajana/thisiskhajana.github.io.git
cd thisiskhajana.github.io

# Option 1: Open directly in browser (simplest)
open index.html

# Option 2: Python local server (recommended for testing)
python -m http.server 8000
# Visit http://localhost:8000

# Option 3: Using VS Code Live Server
# Install Live Server extension and press Go Live
```

### Deployment

Push to `main` branch for automatic GitHub Pages deployment:

```bash
git add .
git commit -m "Update: Add new content to portfolio"
git push origin main
```

The site goes live automatically in ~2 minutes. No build steps needed!
